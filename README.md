# 🏥 Hospital Management System — Python OOP

A complete Hospital Management System built in Python
demonstrating all core Object Oriented Programming concepts.

## 🔧 OOP Concepts Covered
- ✅ Classes & Objects
- ✅ Inheritance (Single + Multi-level)
- ✅ Encapsulation (Private Attributes)
- ✅ Polymorphism
- ✅ Class Variables
- ✅ Static Methods
- ✅ Multiple Inheritance
- ✅ isinstance() Function

## 🏗️ Class Structure
Person (Base Class)
├── Patient
└── Staff
    ├── Doctor
    │    └── Surgeon (Doctor + Operator)
    └── Nurse

## 🚀 How to Run
python project_hospital.py

## 📊 Features
- Patient record management
- Doctor & Nurse management
- Surgeon via Multiple Inheritance
- Hospital statistics
- Working hours validation
- Staff type verification
